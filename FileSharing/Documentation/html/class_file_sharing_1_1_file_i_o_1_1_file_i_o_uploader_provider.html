Detailed Description

Provides methods for uploading to FileIO

FileIO deletes files after 2 weeks in default. FileIO links can only be used once after that the files is deleted

Uploads a the bytes to FileIO and gives links back for the file

Parameters
inputsUploads all arrays and returns an enumerable with all the download links
sleepTimeBetweenDownloadsMSThe amount of time the method should sleep the thread after each download in milliseconds. Default 1000Ms. The default should be enough. Change this only if you have problems with uploading
Returns
Returns the responses for the uploads. To get the download link Use the FileIOJson.FromJson(string) to get a class that has that property

The method waits 1s after uploading a file. this is so there is a smaller chance of the server returning success:false.

This example shows how to upload the files and then download them again

FileIOUploaderProvider uploaderProvider = new FileIOUploaderProvider();
//Upload the test.jpg after splitting it up in 50kb chunks
var links = uploaderProvider.Upload(FileSplitter.SplitFile("test.jpg", 51200)).ToList();
//download the test.jpg
var splitfile = new FileIODownloadProvider().Download(links);
Exceptions
UploadFailedExceptionThrows UploadFailedException if an upload is unsuccesful
